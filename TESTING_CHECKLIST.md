# D8-LPA Testing Checklist - Quick Reference

Print this out or keep it handy while testing!

---

## Before You Start
- [ ] Have your test account credentials
- [ ] Have ~45-60 minutes available
- [ ] Have the feedback form ready
- [ ] Open the site on a desktop/mobile device

---

## Main Features to Explore

### Account & Profile
- [ ] Sign up (or login with test account)
- [ ] Complete profile questions
- [ ] Upload profile photo
- [ ] Update settings
- [ ] Check notification preferences

### Browsing & Matching
- [ ] Browse profiles
- [ ] Use filters (gender, district, etc.)
- [ ] Like/unlike profiles
- [ ] View full profile of someone
- [ ] Check your matches

### Communication
- [ ] Send a message
- [ ] View message history
- [ ] Check notifications

### Events
- [ ] Browse events
- [ ] View event details
- [ ] See event categories and colors

### Design & Usability
- [ ] Notice overall design (colors, fonts)
- [ ] Check if buttons are clear
- [ ] Test navigation between sections
- [ ] Look for any broken features

---

## Questions to Keep in Mind

**While exploring, think about:**
- Is this intuitive?
- Would I know what to do without being told?
- Is the design appealing?
- Are there any confusing parts?
- What would make this better?
- Would I actually use this?

---

## Red Flags to Report

- Anything that doesn't work / crashes
- Buttons that don't do anything
- Confusing or unclear features
- Missing information
- Ugly or hard-to-read text
- Slow loading
- Broken photos

---

## Tips for Giving Good Feedback

✅ **Be specific** - "The message button is hard to find" (good) vs "It's confusing" (not helpful)

✅ **Be honest** - We want real feedback, not just compliments

✅ **Give context** - "I was trying to [action] and didn't know how" 

✅ **Suggest solutions** - "Maybe add a [feature] to make this easier"

✅ **Compare if you know** - "It's like Facebook but with [differences]"

---

## Time Breakdown (Optional)

- Profile Setup: 5-10 min
- Browse: 5-10 min
- Messaging: 3-5 min
- Events: 3-5 min
- General exploring: 10-15 min
- Feedback form: 10-15 min

**Total: 45-60 minutes**

(Feel free to spend more or less time on any section!)
